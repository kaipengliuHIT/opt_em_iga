# Transfer Validation Note

## Purpose

This note records what can already be claimed about the current edge-oriented
transfer operator `P`, and what cannot yet be claimed.

The immediate motivation is to clarify how the current findings may be used in
a future paper without overstating the present level of validation.

## Current Experimental Setting

The transfer operator is tested in a two-level coarse-space setting for a
single-patch Maxwell IGA problem:

- main space: spline/NURBS `H(curl)` space in MFEM
- transfer: edge-oriented auxiliary-space prolongation `P`
- coarse operator: Galerkin projection

`A_c = P^T A P`

This is currently referred to in code as the edge-based Galerkin coarse-space
prototype.

## Main Observation

When the coarse operator is taken as the Galerkin projection `P^T A P`, the
resulting coarse correction is extremely strong on the tested single-patch
problems.

In several representative cases, the outer GMRES iteration count drops from
tens or hundreds of iterations to essentially one coarse-corrected Krylov step.

This behavior is robust across multiple tested source types, not only for a
single smooth Gaussian source.

## What This Validates

The current experiments provide strong evidence for the following statement:

> The present edge-oriented transfer operator `P` captures the dominant
> low-energy / near-null error components of the tested spline `H(curl)`
> Maxwell discretizations well enough that, when paired with the ideal
> Galerkin coarse operator `P^T A P`, it yields near-ideal coarse correction.

Equivalently:

- `P` is not arbitrary noise
- `P` is not fundamentally incompatible with the IGA `H(curl)` space
- the coarse space generated by `Range(P)` is highly relevant to the hard error
  modes of the Maxwell IGA system

This is already a meaningful and publishable intermediate finding if stated
carefully.

## What This Does Not Yet Validate

The current experiments do **not** yet prove any of the following:

1. that `P` satisfies the desired commuting property in a rigorous discrete
   de Rham sense
2. that `P` is already the final transfer operator for the intended Yee-based
   auxiliary-space preconditioner
3. that an independently constructed reference-space Yee Maxwell operator
   `A_Y` will perform as well as the Galerkin coarse operator `P^T A P`
4. that the full intended auxiliary preconditioner

   `M^{-1} = S^{-1} + P A_Y^{-1} P^T`

   has already been validated

In particular, the present Galerkin coarse operator should be interpreted as an
idealized coarse-space reference or upper bound, not as the final Yee
auxiliary-space operator.

## How This Should Be Framed in a Paper

The current result can be stated in paper-ready language along the following
lines:

### Safe claim

> Using the proposed edge-oriented transfer operator together with the Galerkin
> coarse operator `P^T A P`, we observe near-ideal two-level coarse correction
> for the tested single-patch spline `H(curl)` Maxwell systems.

### Interpretation

> This indicates that the transfer operator captures the dominant low-frequency
> and near-null error components of the isogeometric Maxwell discretization.

### Necessary limitation

> However, this only validates the transfer in the Galerkin coarse-space
> setting. It does not yet establish that an independently constructed
> NURBS-mapped Yee auxiliary operator `A_Y` will reproduce the same behavior.

## Research Role of the Current Result

The present result should be viewed as a transfer-validation milestone.

It serves three purposes:

1. It confirms that the edge-oriented coarse space itself is promising.
2. It provides an upper-bound reference for what a high-quality edge-based
   auxiliary correction can achieve.
3. It defines the next research question clearly:

   how closely can an independently assembled Yee auxiliary operator `A_Y`
   approximate the ideal Galerkin coarse correction generated by `P^T A P`?

## Immediate Next Step

The next step is therefore not to further optimize the Galerkin coarse operator
itself, but to construct a true `edge_yee_proto`:

- keep the transfer `P`
- replace `P^T A P` with an independently assembled reference-space Yee
  auxiliary Maxwell operator `A_Y`
- compare `P A_Y^{-1} P^T` against the Galerkin benchmark

This will directly test whether the original theoretical auxiliary-space idea
can approach the currently observed ideal coarse-space behavior.
